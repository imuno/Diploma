v1.4.0 Update Notes

In this update the file structure was fully improved in order to provide more clean and simpler file and code organization. This update includes following key changes:

  1. default demo ID has been renamed to demo1.
  2. The source src folder has been fully reconstructed to provide more optimizied file structure for SASS, JS and Media files.
  3. The assets folder has been slightly reconstructed to separate the assets versions per working demo.
  4. The only change in the code is with the assets paths so refer to this version and update the assets paths accordingly. However no changes has been made in the actual HTML, SASS or JS code.
