Sparrow includes a smooth scrolling plugin which allows you to animate the page's scrolling to elements within the page. The markup snippet below demonstrates:

Scroll to the #example-section below 
<a href="#example-section" data-fancyscroll="true" data-offset="60">
	example section
</a>
<section id="example-section">
	...
</section>

The above link will scroll to this section.
